# ETF Finder

A modern ETF discovery and portfolio building tool built with React + Vite.

## Features

- **Smart Filters** — Filter by sector, industry, risk category, and performance ranges
- **Interest-Based Search** — AI-powered keyword matching to find ETFs by topic (e.g., "AI", "clean energy", "retirement")
- **Performance Dropdowns** — Preset and custom range filters across 9 time periods (1D to 3Y)
- **Package Builder** — Select and compare multiple ETFs side by side
- **Expandable Details** — Click any ETF row for full details including metrics, performance grid, and tags
- **Package Analytics** — Aggregate stats, average performance, and sector exposure for your selected ETFs
- **Disclosure Compliance** — Built-in disclaimer footer and full disclosure modal

## Tech Stack

- React 19
- Vite
- Pure CSS (no external UI libraries)
- Pre-computed interest mappings (zero API cost)

## Getting Started

```bash
npm install
npm run dev
```
